FPGA Retro Console

Retro console built on the the DE1-SoC FPGA board hardware. This project simulates retro style games using hardware rendering and computing like early consoles. This includes a ROM memory for games, a sprite system, a hardware physics engine, SVGA output, double video buffering and much more. Games included are Pong and Dino (Chrome's offline dino game).

Features

  • ROM memory for hard assests like the ASCI character set.
  • Sprite system with a 15 bit RGB value and 1 bit alpha value allowing for overlapping of sprites.
  • A simple and a complex hardware based physics engine for low latency and realistic gameplay.
  • SVGA protocol output with full 800x600 resolution at 60Hz.
  • Double video buffering to prevent screen tearing and to give a cleaner output signal.
  • SDRAM to give enough capacity for the double video buffering at 800x600, 60Hz resolution.
  • PS2 keyboard inputs for user control.

Development Skills

  • Hardware design.
  • Game design and optimization.
  • Various protocols and standards.

Development Tools

  • Quartus Prime.
  • Verilog.
  • DE1-SoC board.

TTC Navigator

Mobile transit app dedicated to the Toronto Transit Commission (TTC) focused on displaying information about station connections between different routes and stations on bus, streetcar and subway routes. This app does not try to compete with existing transit apps like Apple Maps and Google Maps but rather is intended for transportation enthusiasts who want to see how the system is connected.

Features

  • Station search feature to find nearby stations, the routes that operate through that stations and the next 3 vehicles for each route in all directions.
  • Station map feature displays all stations and the appropriate vehicle type(s).
  • Route search feature to look up all bus, streetcar and subway routes with all stations that are found on the route.
  • Settings feature allows for a ton of customization to the behaviour and visual aesthetic.

Development Skills

  • Multi-threaded Programming.
  • API networking.
  • Graphics design (UI/UX).

Development Tools

  • Dart programming language.
  • Flutter framework.
  • OpenStreetMap API.
  • TTC API.
  • Unqimo API.

Numpad Keyboard

The Numpad Keyboard project is a mechanical numpad and multifunction keyboard for the purpose of learning the basics of PCB design and soldering. With VIA and QMK capabilities, this numpad can be used for any inputs like a macro pad, and includes flexibility for different layouts with the hotswap sockets.

Features

  • Multiple layout options using a stack 2U and 2x1U keys.
  • Reprogrammable ATmega32uA microchip using QMK and VIA software.
  • Kailh hotswap sockets allow for quick interchanging of mechanical key switches.

Development Skills

  • Circuit design.
  • User input analysis.
  • Board assembly including soldering.

Development Tools

  • Subset of KiCad tools.
  • Soldering equipment.
  • VIA and QMK.

Dynamic Paddle

A short skill based game, created as a challenge to build a game using on HTML, CSS and Javascript. Keep the ball from falling below the screen using only a paddle that dynamically moves as you move your cursor.

Features

  • Realistic physics coupled with the dynamic paddle creates fast paced gameplay which is challenging.
  • Simplistic styling allows for ease of use while looking aesthetically pleasing.
  • Frame rate compensation ensures that weaker computer systems can have a similar experience.
  • Lost focus detection makes sure that users who drag the cursor offscreen or switch tabs, the game is paused.

Development Skills

  • Game design.
  • Web design, HTML and CSS.
  • Javascript programming language.
  • Graphics design (UI/UX).

Development Tools

  • HTML5 Boilerplate VSCode extension (Quick set up for initial boilerplate code).
  • Live Server VSCode extension (Allows easy development including hot reloading, and various browser support to ensure continuity).
  • IntelliSense for CSS class names in HTML VSCode extension (Easily identify and locate the class names code).

Circle Serenity
